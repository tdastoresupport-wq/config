Cài Cấu Hình Web Clip
Thêm Zev Boost vào màn hình chính iPhone của bạn
1
Tải file ZevBoost.mobileconfig từ repository GitHub của Zev Boost về thiết bị iOS.
2
Truy cập Cài đặt → Đã tải về Hồ sơ và nhấn Cài đặt để xác nhận cài profile.
3
Nếu cần ký profile tự ký, sử dụng vsign.me hoặc công cụ OpenSSL để tạo chứng chỉ hợp lệ.
4
Sau khi cài đặt thành công, biểu tượng Zev Boost sẽ xuất hiện trên màn hình chính iOS dưới dạng Web Clip toàn màn hình.
5
Khởi động Web Clip và trải nghiệm dashboard tối ưu hóa OB54 PRO của bạn!
